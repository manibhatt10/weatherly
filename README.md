# Weatherly

A simple Django weather application.

## Setup

1. Install Django:
   ```
   pip install django
   ```

2. Run migrations:
   ```
   python manage.py migrate
   ```

3. Start the development server:
   ```
   python manage.py runserver
   ```

4. Open your browser and visit: [http://127.0.0.1:8000](http://127.0.0.1:8000)
